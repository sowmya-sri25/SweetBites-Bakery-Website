# 🍰 SweetBites – Bakery Website
SweetBites is a simple bakery-themed webpage built using HTML and CSS.  
It displays bakery products, pricing information, features of the bakery, and a contact form.
## 📌 Project Overview
This project is created to practice basic web development concepts such as:
- HTML page structure
- Semantic elements
- Image integration
- Tables and forms
- Basic CSS styling
- Layout using div containers
## ✨ Features
- 🧁 Bakery product showcase section
- 📋 Product price table
- ⭐ "Why Choose Us" feature list
- 📖 About Us section
- 📞 Contact form
- 🦶 Footer section
## 🛠️ Technologies Used
- HTML5
- CSS3
## 🎯 Learning Outcome
Through this project I practiced:
- Structuring a webpage using HTML
- Styling elements using CSS
- Creating product cards
- Building tables and forms
